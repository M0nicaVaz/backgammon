## Setup

**Clone the project and access the folder**

```bash
git clone https://github.com/M0nicaVaz/backgammon.git && cd backgammon
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm or yarn install

# Run the web server
$ npm run dev
```
